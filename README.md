# Schola-System
Sistem Informasi Pendaftaran Beasiswa Berbasis Web untuk Transparansi dan Efisiensi Administrasi Pendidikanuntuk mengelola informasi beasiswa dan pendaftaran online secara terstruktur. Fitur: Dapat membuat akun, dapat melihat daftar beasiswa, mendaftarkan beasiswa, dapat menambah dan mengubah beasiswa, dapat update status pendaftaran.
